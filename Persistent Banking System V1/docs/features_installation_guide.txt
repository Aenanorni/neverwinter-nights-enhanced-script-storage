
Banking System Features & Installation Guide
-------------------------------------------
Version: 1.0

Features:
- Persistent item and gold storage
- NPC banker with dialogue branching
- Admin tools for DMs
- SQL-based persistence
- Modular script architecture

Installation Summary:
- Import scripts via Script Editor
- Assign retrieve/store scripts to chest OnOpen/OnClose
- Assign conversation script to banker NPC OnConversation
- Use schema.sql to create database tables
- Configure admin tools for DM use
